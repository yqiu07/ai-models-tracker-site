## Report 文件夹说明

本文件夹存放模型追踪任务的**更新报告**，每次对 `Object-Models-Updated.xlsx` 进行批量更新后，在此生成一份报告文档。

---

### 文件命名规范

```
update_report_YYYYMMDD.md    — 每次更新的成果报告
```

### 报告内容模板

每份报告应包含：

1. **更新概览** — 本次新增/更新的模型数量、数据来源
2. **数据来源覆盖** — 各来源渠道的提取情况
3. **数据质量统计** — 各字段的覆盖率
4. **可视化图表** — 按公司/类型/国内外等维度的分布
5. **已知问题与待人工核实项** — 需要人工确认的内容
6. **后续建议** — 下一步应做什么
